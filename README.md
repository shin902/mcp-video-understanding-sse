# MCP Video Understanding

## 概要
このプロジェクトは、Geminiモデルを使用したビデオ理解システムのMCPサーバーです。

## 環境要件
- Node.js
- TypeScript
- Cloudflare Workers

## セットアップ手順
1. 依存関係のインストール
```bash
npm install
```

2. 環境変数の設定
`.env`ファイルに必要な環境変数を設定してください。

## 開発
- ビルド: `npm run build`
- テスト: `npm test`
- 開発サーバー: `npm run dev`

## デプロイ
`npm run deploy`
